
# PixelBoy

### Keyboard mapping

| Gameboy   | PC        |
| ------------  | ------------   |
| A         | C         | 
| B         | V         | 
| Select    | Space     |
| Start     | Enter     |
| Right     | A Key     |
| Left      | D Key     |
| Up        | W Key     |
| Down      | S Key     |


### Running 

Build and run using
```sh
  make release
```
